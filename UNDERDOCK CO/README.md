## ⚙️ Personalizaciones Recomendadas

### En `index.html`:

**Cambiar número WhatsApp (línea ~150)**:
```javascript
onclick="document.getElementById('servicios').scrollIntoView({behavior: 'smooth'})"
```

**Cambiar enlace Instagram (línea ~400)**:
```html
<a href="https://instagram.com/tu-cuenta" target="_blank">@tu-cuenta</a>
```

**Cambiar ubicación (línea ~410)**:
```html
<p>Tu Calle, 123<br>Tu Ciudad, País</p>
```

**Cambiar número WhatsApp en contacto (línea ~430)**:
```javascript
window.open(`https://wa.me/TU-NUMERO?text=${encodedMessage}`, '_blank');
```

### En `styles.css`:

Todos los colores están en variables CSS `:root` (líneas 1-12). Modifica valores según necesidad.
